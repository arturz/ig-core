## Environment variables
- `JWT_SECRET`
- `LOGIN` - for dev instagram bot
- `PASSWORD` - for dev instagram bot

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
